# 2. 자료 올리기
# push전에는 pull이 있다 : git에 자료를 올리고 당겨오고를 하는것
# git pull origin master : 나의 연결된 깃에 자료 당겨오기
# git status : 나의 폴더와 깃 repository에 연결 상태 확인하기
# git add . : 뒤에 한칸띄고 . 을 붙이면 모든 변경된 자료 올리기가됨
# git commit -m "메세지" : 나의 레포지토리에 자료 올릴때 메세지를 입력해서 올릴수 있음
# git push origin master : 실제로 자료를 올리는 명령어
